# Project Title
Escape the Cursed Temple

## Description
This is a console-based adventure game developed in C. The player explores a 20Ã—20 temple grid, collecting keys, avoiding traps,
and using portals to reach hidden areas. The goal is to escape within 90 seconds while managing health and making strategic decisions.

## Features
- Grid-based movement using W/A/S/D keys
- Mystery boxes with random outcomes (key or trap)
- Health system and game-over condition
- Exit gate appears after collecting all keys
- Portals for teleportation to secret zones
- Real-time countdown using GetTickCount()
- Modular code structure with functions and arrays.
  
## Technologies Used
- Programming Language: C
- Libraries: stdio.h, stdlib.h, time.h, windows.h
  
## How to Run
- Open the project in a C-compatible IDE (e.g., Code::Blocks, Dev-C++, Visual Studio).
- Ensure the compiler supports windows.h.
- Compile and run the main C file.
- Follow the on-screen instructions to play.
